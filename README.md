# 价值观超级大冒险

这是一个原创横版跳跃 H5 小游戏。玩家操控负鼠穿过平台、弹簧、小障碍和更长的关卡路线，收集五枚价值观徽章。熟悉路线后，完整通关大约 2 分钟左右。

- 客户第一
- 团队协作
- 拼搏进取
- 持续学习
- 创新创业

## 运行方式

直接打开 `index.html` 即可体验。

也可以启动本地服务：

```bash
node server.mjs
```

然后访问 `http://127.0.0.1:4173`。

## 操作

- 方向键 / A、D：移动
- 空格 / W / 上方向键：跳跃
- R：重新开始
- 手机或平板可使用页面底部的方向按钮

## 文件说明

- `index.html`：页面入口
- `styles.css`：界面、弹窗、徽章与移动端按钮样式
- `app.js`：平台跳跃玩法、关卡、碰撞、收集与结算逻辑
- `server.mjs`：本地预览服务
