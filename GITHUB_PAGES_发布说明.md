# GitHub Pages 发布说明

这个目录可以直接上传到 GitHub 仓库根目录。

发布步骤：
1. 新建或打开一个 GitHub 仓库，例如 value-adventure-game。
2. 上传本目录中的所有文件和 assets 文件夹。
3. 进入仓库 Settings > Pages。
4. Source 选择 Deploy from a branch。
5. Branch 选择 main，目录选择 /root。
6. 保存后等待 GitHub Pages 生成访问链接。

建议仓库名：value-adventure-game
访问地址通常是：https://你的用户名.github.io/value-adventure-game/
